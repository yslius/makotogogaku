無料でヒートマップ解析を行うプラグインです。ユーザーの行動を可視化して、コンバージョンアップ！

== Description ==

このプラグインは無料ヒートマップツールであるUserHeatをWordPressとワンステップで連携するためのプラグインです。

UserHeatは「ユーザーが画面のどこを見ているか」「どこをクリックしたか」などのヒートマップ解析が可能で、WEBサイトにアクセスしたユーザーの行動を可視化するアクセス解析ツールです。
30秒程度の登録作業だけで無料で利用することが可能です。


◆5種類のヒートマップ
・熟読エリアヒートマップ
・マウストラッキング
・クリックエリアヒートマップ
・終了エリアヒートマップ
・離脱エリアヒートマップ

◆PC・スマートフォンに対応
PC・スマホそれぞれのヒートマップ分析が可能です。

◆EFO（エントリーフォーム最適化）機能
お問い合わせフォームや登録フォームのどこで、ユーザーが離脱しているかをグラフィカルに分析します。

◆検索キーワード分析機能
GoogleSearchConsoleと連携することで、検索流入キーワードの分析が可能です。


<a href="https://userheat.com">UserHeatについてより詳しく見る</a>


== Installation ==

1. プラグインをインストールします。
2. UserHeatのサイトに移動し、URL・メールアドレス・パスワードなどを入力して登録します。
3. 発行されたサイトIDをプラグイン上で登録します。
4. 解析が始まるのでしばらくすると、訪れたユーザーの解析結果が見られるようになります。
